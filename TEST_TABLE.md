# LCUMod 完整测试表

## 启动准备
- [ ] 启动 Python 后端: `cd backend && python main.py`

## 注意事项

1. **挖掘需要生存模式** — 创造模式不会破坏方块
2. **物品收集需要** 在热栏有正确工具
3. **自动进食** 需要食物在热栏
4. **容器交互** 需要玩家面向容器方块
5. **聊天冷却** 1.5s 防止服务端反刷踢出
